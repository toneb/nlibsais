# nlibsais
.NET cross-platform bindings for [libsais](https://github.com/IlyaGrebnov/libsais).

## Supported runtimes
* Windows (x86, x64, amd64)
* Linux (x64, arm, arm64)
* MacOS (x86_64, arm64)
